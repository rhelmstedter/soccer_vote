# Directions
Both methods assume you have python installed on your machine and you have cloned the repo.

## Playwright (Recommended)
1. Pip install playwright
2. Run "playwright install"
3. Run the script `vote.py`

## Pyautogui
1. Clone repo
2. pip install pyautogui
3. pip install pillow
4. Take a screenshot of the player's name. Save it in the repo as 'player.png'.
5. Take a screenshot of the vote button. Save it in the repo as 'vote.png'.
6. Run the script `vote2.py`.
